# Prompt Engineering para Iniciantes

Guia prático para aprender a escrever prompts melhores e usar inteligência artificial de forma mais clara, útil e eficiente no dia a dia.

## Sobre o projeto

Este repositório reúne um estudo introdutório sobre boas práticas e engenharia de prompts para iniciantes. O foco é aprender, testar e reaproveitar prompts para tarefas como resumo de textos, explicações didáticas, revisão de conteúdo e organização de informações.  
As boas práticas mais recorrentes nas fontes escolhidas incluem contexto claro, instruções específicas, definição de formato de saída e melhoria iterativa do prompt.  
Esses princípios aparecem com frequência em guias como o do Google Cloud, MIT, OpenAI e Prompting Guide.

## Objetivos

- Entender o que é um prompt e por que sua estrutura influencia a qualidade da resposta.
- Aprender a escrever prompts mais claros, específicos e úteis.
- Testar modelos práticos de prompts para estudo, resumo e revisão.
- Criar uma base reutilizável de prompts para uso pessoal.
- Aplicar esse aprendizado em ferramentas como NotebookLM, ChatGPT, Gemini, Claude e Perplexity.

## Testes de prompts

Use estes testes como laboratório prático.

### 1. Explicação para iniciantes

```text
Explique o que é engenharia de prompts para um iniciante, em português do Brasil, com linguagem simples, 5 tópicos e 2 exemplos práticos.
```

### 2. Resumo de texto

```text
Resuma o texto abaixo em até 8 linhas.
Depois liste 5 ideias principais e crie 3 perguntas de revisão com respostas curtas.
```

### 3. Melhoria de prompt

```text
Analise este prompt, diga por que ele está fraco e reescreva em uma versão melhor:
[cole aqui o prompt]
```

### 4. Estudo de vídeo ou transcrição

```text
Com base na transcrição abaixo, faça um resumo, destaque os pontos principais e crie um miniguia de estudo em tópicos.
```

### 5. Comparação entre duas fontes

```text
Compare as duas fontes abaixo sobre boas práticas de prompts.
Mostre semelhanças, diferenças e recomendações práticas em tópicos curtos.
```

## Miniguia de estudo

### Etapa 1 — Fundamentos
- Ler um texto introdutório sobre prompt engineering.
- Identificar os princípios centrais: contexto, clareza, especificidade e formato.

### Etapa 2 — Observação
- Comparar exemplos de prompts fracos e prompts melhorados.
- Observar como pequenas mudanças alteram a resposta da IA.

### Etapa 3 — Prática
- Testar 3 prompts próprios.
- Reescrever cada um com mais contexto e objetivo.
- Comparar os resultados.

### Etapa 4 — Reutilização
- Guardar os melhores prompts como modelos.
- Adaptá-los para estudo, revisão, resumos e organização pessoal.

```text
4. Reescrever exemplos fracos em versões melhores.
5. Criar 3 perguntas de revisão ao final.

Tema ou fonte a analisar:

```
